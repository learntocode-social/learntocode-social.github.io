Frequently Asked Questions

How much do I need to pay for this program? Is it paid or free?

LearnToCode.Social is completely free with no strings attached. You can participate in our community and learn and you don't have to pay anything.

When does the program start and end?

There are no start and end dates. You can join at any time and start learning. You can pick a learning path of your choosing and learn at your own pace.

Where do I need to come? Is this program online?

LearnToCode.Social is completely online. Everything we do, we do it in our online community.

Will I get a Job?

There are two events that you can apply for. RSPCT and Code for India Fellowship program.

To be eligible, you need to:

  • Get an Algorithms Badge on HackerRank
  • And you need a Web or Mobile App up on GitHub that interacts with the user, saves data to a server, works offline and displays local language text.
