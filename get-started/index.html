Programming is learned by writing programs

- Brian Kernighan

Getting started in Software Development and programming is easy due to the nature of Software and the huge amount of high quality learning resources available on the internet. But the journey is not simple. It takes time and regular effort to get good at programming. Realistically you should be looking at around 6 months to around 1 year of practice to get past the beginner level.

With that said, read the following articles first before starting anything.

  • Read Peter Norvig's (Director of Research at Google) Teach Yourself Programming in Ten Years
  • Read Which is the best programming language / What should I learn?
  • Read Tips and Techniques to keep in mind when Learning Programming
  • Read How to Ask

Programming Learning Path

If you are only interested in getting started in programming and not concerned with learning to build Web or Mobile Apps then:

  1. Complete How to think like a Computer Scientist
  2. Complete Automate the Boring Stuff with Python
  3. Build a project from what you have learned thus far or apply what you have learned in your day to day life or work

Web Learning Path

  1. Watch: What I wish someone had told me when I started Web Development
  2. Learn the basics of HTML & CSS
    1. Start and complete Codecademy's HTML & CSS track
    2. Complete Codecademy's Make a Website
    3. Read through the first 5 questions of the most frequent questions on Stack Overflow tagged HTML
    4. Read through the first 5 questions of the most frequent questions on Stack Overflow tagged CSS

      • Go through GitHub Bootcamp for an introduction to GitHub
      • Go through GitHub's introduction to GitHub Pages or Mozilla's step-by-step tutorial on publishing your Website
    2. Next, fork and go through the source code of the PyCon India website. This will get you get started in understanding how others write code.
  3. Learn the programming language of the Web: Javascript
    1. Complete Codecademy's Javascript track
    2. Complete Codecademy's jQuery track

  4. Understand Javascript
    1. Complete Eloquent Javascript
    2. Read through the first 5 questions of the most frequent questions on Stack Overflow tagged Javascript
    3. Read through the first 5 questions of the most frequent questions on Stack Overflow tagged jQuery
  5. Learn Python
    1. Complete Codecademy's Python track
    2. Complete How to think like a Computer Scientist
    3. Complete Udacity's Programming foundations with Python
    4. Read through the first 5 questions of the most frequent questions on Stack Overflow tagged Python
    5. Upload the exercises that you complete to your GitHub profile
  6. Learn SQL
    1. Complete Khan Academy's SQL tutorial
    2. Complete Udacity's Introduction to Relational Databases
    3. Read through the first 5 questions of the most frequent questions on Stack Overflow tagged SQL
  7. Learn to build a complete Web App
    1. Complete Udacity's Full Stack foundations
    2. Build a simple project using all you have learned thus far and publish it on GitHub

Android Learning Path

  1. If you are new to Java then start with Intro to Java Programming
  2. Complete Udacity's Developing Android Apps or if you prefer written tutorials, go through Google's Official Android training
  3. Build a simple App and publish it on your GitHub profile

Algorithms and Data structures

It is time to get better at thinking like a programmer and learn the concepts that'll help you get better at designing elegant solutions to problems.

  1. Complete Khan Academy's Algoirthms course
  2. Get an Algorithms Badge on HackerRank. You need to score at least 20 points in each of the following sub-domains to get an Algorithmic Badge on HackerRank: Arrays and Sorting, Dynamic Programming, Graph Theory and Combinatorics
