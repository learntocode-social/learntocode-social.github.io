Getting Started

The first step in Learning to Code begins with creating accounts in the following websites:

  • Create an account on Codecademy
  • Create an account on HackerRank
  • Create an account on Stack Overflow
  • Create an account on GitHub

If you want to get started in programming or if you want to make Websites

Start and complete Codecademy's HTML & CSS class. Then make a website for yourself and put it on neocities. Search around the internet on what makes a good personal website and take inspiration from your favorite websites on the Web. Once you have completed the above, begin your programming journey with Codecademy's JavaScript class.

If you want to learn Android

If you are new to Java then start with Intro to Java Programming. Then go through Udacity's Developing Android Apps or if you prefer reading, go through Google's Android training. Build an app (however simple it maybe!) and put it on Google's Play Store.

HackerRank

As you start getting comfortable in programming and building software, slowly start solving the Algorithmic challenges on HackerRank. It will help you to think like a programmer and make you better at designing elegant solutions to problems.

You need to score at least 20 points in each of the following sub-domains to get an Algorithmic Badge on HackerRank: Arrays and Sorting, Dynamic Programming, Graph Theory and Combinatorics

Conclusion

Remember, getting started is easy, especially in software development, but the journey is not simple. It takes time and regular effort to get good at programming. Realistically you should be looking at around 3 - 6 months of practice to get past the beginner level.

That is what the Learn to Code Community is here for. The community assists and guides you in learning the skills you need to become a professional software developer. You can ask questions, chat with other programmers and hold discussions. But before that, make sure that you have also gone through our Resources page.
